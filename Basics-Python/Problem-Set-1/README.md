<h2> How it Works </h2>

The Christmas tree will be formed using asterisks ('\*'). The number of stars in each line corresponds to the branch number. For example, if you choose a tree length of 5, it will look like this:

```
      *     # 1 star (Line 1)
     ***    # 3 stars (Line 2)
    *****   # 5 stars (Line 3)
   *******  # 7 stars (Line 4)
  ********* # 9 stars (Line 5)
```
